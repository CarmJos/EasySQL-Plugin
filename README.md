## 指令

插件主指令为 `/easysql` ，所有指令只允许后台执行。

```text
# help
- 查看插件指令帮助。

# version
- 查看当前插件版本与核心库(EasySQL)版本。

# list
- 列出当前所有的数据源管理器与相关信息。

# stats <数据源名称>
- 查看指定数据源的统计信息与当前仍未关闭的查询。
```
